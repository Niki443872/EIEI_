NEW FEATURES:
=============

1. PROGRAM GALLERIES
   - Each program page now has its own image gallery
   - Managed from Admin Panel → "Program Galleries" tab
   - Upload up to 5 images per program
   - Auto-uploads to ImgBB
   - Displays on all 6 program pages

2. TESTIMONIAL GALLERIES
   - Shared gallery for all testimonials
   - Managed from Admin Panel → "Testimonial Gallery" tab
   - Upload up to 5 images
   - Auto-uploads to ImgBB
   - Displays on: index.html, training-development.html, early-intervention.html

ADMIN USAGE:
============

Upload Program Gallery:
1. Go to admin-programs.html
2. Sidebar → "Program Galleries"
3. Select program from dropdown
4. Drag-drop images or click to browse
5. Click "Upload Images"
6. Done!

Upload Testimonial Gallery:
1. Go to admin-programs.html
2. Sidebar → "Testimonial Gallery"
3. Drag-drop images or click to browse
4. Click "Upload Images"
5. Done!
